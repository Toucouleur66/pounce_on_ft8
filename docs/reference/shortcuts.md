# Keyboard Shortcuts

All shortcuts use <kbd>Ctrl</kbd> on Windows/Linux and <kbd>⌘</kbd> on macOS.

## App

| Shortcut | Action |
|---|---|
| <kbd>Ctrl</kbd>+<kbd>M</kbd> | Start Monitoring / Stop all (toggle) |
| <kbd>Ctrl</kbd>+<kbd>R</kbd> | Restart (macOS) |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | Toggle sound alerts |
| <kbd>Ctrl</kbd>+<kbd>,</kbd> | Open Settings |
| <kbd>Ctrl</kbd>+<kbd>I</kbd> | Check for Updates |
| <kbd>Ctrl</kbd>+<kbd>L</kbd> | Show Active Users window |

## View

| Shortcut | Action |
|---|---|
| <kbd>Ctrl</kbd>+<kbd>C</kbd> | Compact View |
| <kbd>Ctrl</kbd>+<kbd>A</kbd> | Show All Messages |
| <kbd>Ctrl</kbd>+<kbd>F</kbd> | Show / hide Filters |
| <kbd>Ctrl</kbd>+<kbd>G</kbd> | Grid Monitoring (map) |
| <kbd>Ctrl</kbd>+<kbd>W</kbd> | Clear Filters |
| <kbd>Ctrl</kbd>+<kbd>K</kbd> | Clear rows from Table |
